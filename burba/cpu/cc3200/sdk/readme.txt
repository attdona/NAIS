================================================================================
CC3200 SDK v1.3.0 release
================================================================================

The CC3200 SDK hosts building blocks viz. Peripheral Drivers,  SimpleLink Host,
MCU Power Management Framework and many sample applications covering Wi-Fi
features, Networking and MCU Peripherals. Appropriate documentation needed to 
use the CC3200 Wireless MCU, Internet-on-a-chip™ Solution is also included in
the package.CC3200 SDK is structured to accompany CC3200LAUNCHXL LaunchPad kit
and supports multiple IDE options Viz.  Code Composer Studio, IAR, GCC etc.. 


Getting started
================================================================================
Follow the instructions at <http://www.ti.com/lit/pdf/swru376> or 
'<sdk-installation_location>\cc3200-sdk\docs\
CC3200-Getting_Started_Guide.pdf' for setting up your development 
environment, downloading and installing the needed software/tools 
and finally, executing your first project!
